# Curso de Introducción al Desarrollo Web — UNSA

## Página sobre el número PI

## Descripción
Es una plataforma web educativa diseñada para desmitificar el número PI (π) a través de una experiencia interactiva y multidimensional. Este proyecto académico implementa una arquitectura web completa que permite a los usuarios descubrir la fascinante historia de esta constante matemática, explorar sus aplicaciones prácticas, y participar en actividades lúdicas que demuestran su importancia en la ciencia y la vida cotidiana. El sistema integra contenidos educativos con funcionalidades interactivas, garantizando una experiencia de aprendizaje envolvente y accesible para diferentes niveles de conocimiento matemático.
## Lenguajes utilizados
- HTML - Para la estructura y contenido de la página web
- CSS - Para los estilos y diseño responsivo
- JavaScript - Para la interactividad y funcionalidades dinámicas
- Python - Para la lógica del servidor y procesamiento de datos
- SQL - Para la gestión y consulta de la base de datos
