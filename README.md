# Curso de Introducción al Desarrollo Web — UNSA

## Página sobre el número PI

## Descripción
Esta página es sobre el número PI, habla sobre su historia, datos curiosos y aplicaciones, tambien incluye apartados interactivos, como:
- Un juego que consiste en escribir la mayor cantidad de los valores de PI
- Una simulacion que permite acercarse al valor de PI

## Lenguajes utilizados
HTML - Para la estructura y contenido de la página web
CSS - Para los estilos y diseño responsivo
JavaScript - Para la interactividad y funcionalidades dinámicas
Python - Para la lógica del servidor y procesamiento de datos
SQL - Para la gestión y consulta de la base de datos
